# Bundled fonts

**EB Garamond** — © 2017 Georg Duffner (https://fonts.google.com/specimen/EB+Garamond)
Licensed under SIL Open Font License v1.1 (see `OFL.txt`).

Used for vector text rendering in book covers. Chosen for:
- Classical proportions suitable for literary fiction
- Full Unicode support (en-dash, em-dash, smart quotes, curly apostrophes)
- Italic + bold + bold-italic variants for typographic hierarchy
- Free to bundle and redistribute under OFL

Files:
- `EBGaramond-Regular.ttf` — body text, chapter paragraphs
- `EBGaramond-Italic.ttf` — quotes, taglines, bylines
- `EBGaramond-Bold.ttf` — titles, section labels
- `EBGaramond-BoldItalic.ttf` — emphasized headings

See `OFL.txt` for the full license.
